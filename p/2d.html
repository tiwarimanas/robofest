2D Game Project

From..

Neel, Vinay, and Daksh - Class 7

Application

The 2D Game Project brings the joy of gaming to life using Arduino and an LCD display. Inspired by the popular Google Chrome Dino game, this project allows players to control a character on the LCD screen using a push button. With obstacles to dodge and points to score, the game provides entertainment and challenges for players of all ages. The project demonstrates the potential of Arduino and simple electronics to create engaging and interactive experiences.

System Behind

The project utilizes an Arduino microcontroller connected to an LCD display and a push button. The microcontroller runs the game logic, controlling the movement of the character on the LCD screen based on input from the push button. Players can jump to avoid obstacles by pressing the button at the right time. The game keeps track of the score and increases the difficulty as the player progresses. This integration of hardware and software creates a fun and immersive gaming experience reminiscent of classic 2D platformers.
