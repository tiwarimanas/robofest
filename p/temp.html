Temperature and Humidity Sensor Project

From..

Manu Tiwari - Class 8

Application

The Temperature and Humidity Sensor project models the functionality of an ice factory by monitoring environmental conditions. Using a DHT11 sensor, the system provides real-time data on the current temperature and humidity levels. When the temperature exceeds 37 degrees Celsius, an alert is displayed on the OLED display, signaling a potential rise in temperature that could affect the ice storage conditions.

System Behind

The project utilizes a DHT11 sensor to collect environmental data. This data is then processed by a microcontroller, such as Arduino, which determines whether an alert needs to be triggered based on predefined thresholds. If the temperature exceeds 37 degrees Celsius, the microcontroller sends a signal to the OLED display to display an alert message. Otherwise, the real-time temperature and humidity readings are displayed on the OLED display, providing valuable information for monitoring the ice factory conditions.
