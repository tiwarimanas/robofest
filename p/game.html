Application

The Wire Loop Game project is a fun and interactive electronic game that challenges players' hand-eye coordination. Using LEDs and Arduino, players navigate a looped wire along a predefined path without touching the sides. If the loop touches the sides, LEDs light up to indicate a failed attempt. This project demonstrates the integration of electronics and gaming to create an engaging and entertaining experience for users of all ages.

System Behind

The project utilizes LEDs and Arduino to create a wire loop game. LEDs are placed along the path of the wire loop, and sensors detect when the loop makes contact with the sides. When contact is detected, the Arduino triggers the corresponding LEDs to light up, indicating a failed attempt. This system provides real-time feedback to players and enhances the gaming experience by adding visual cues to the gameplay.
